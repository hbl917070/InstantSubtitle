# 即時字幕 InstantSubtitle

有時候只是要介紹一些簡單的操作，但是卻又不能沒有字幕說明，然後後製影片加字幕是一件相當麻煩的事情<br>
為了解決這個窘境，於是開發了這個操作極簡單的「即時字幕」<br>
只要事先將要顯示的字幕輸入到程式，就能夠在螢幕即時的顯示出字幕，並且將其朗讀出來
